# quickstart express app

```
$ npm install
$ ./bin/www
// open http://localhost:3000
```

## features

* Autobuild stylus css
* Autopackage js with browserify
* Hash static resources for better caching
* ENV loading in development
* Simple and extendable logging

## modules

* express
* hbs
* stylish
* enchilada
* taters
* localenv
* book
* httperrors
